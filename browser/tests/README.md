# Browser test bundle directory

This directory is intentionally named `tests` (plural). Do **not** rename
to `test` (singular) — Node's `node --test` discovery treats any directory
named exactly `test` as a test source root and would auto-execute the
bundled artifact under this directory, breaking the no-argument
`node --test` workflow (e.g. when run from an IDE's "run all tests"
action).

The bundle (`bundled.js`) is the rollup output of the test sources,
intended to be loaded from `../tests.html` via
`<script src="./tests/bundled.js">`. It is regenerated by `make` and
is excluded from `.gitignore`.
